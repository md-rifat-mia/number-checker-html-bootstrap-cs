# 🔢 Number Checker

A simple and responsive **Number Checker** web application built with **HTML, Bootstrap, CSS, and JavaScript**. This project allows users to enter a number and instantly validate whether the input is a valid number.

## 📖 About the Project

The **Number Checker** is a beginner-friendly JavaScript project that validates user input. It checks whether the entered value is a valid number and displays the appropriate result.

This project is useful for practicing:

- HTML Form Handling
- Bootstrap Components
- JavaScript Input Validation
- DOM Manipulation
- Responsive Web Design

---

## ✨ Features

- ✅ Clean and responsive UI
- ✅ User-friendly input field
- ✅ Validates number input
- ✅ Instant result display
- ✅ Built using pure JavaScript

## 💻 How to Run

1. Clone the repository

```bash
git clone https://github.com/md-rifat-mia/number-checker-html-bootstrap-cs.git
```

2. Open the project folder

```bash
cd number-checker-html-bootstrap-cs
```

3. Open `index.html` in your browser.
